## Local Development

Clone the repository:

```bash
git clone https://github.com/Bhvya11singh/introduction_portfolio.git
```

Navigate to the project directory:

```bash
cd introduction_portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
